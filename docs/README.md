# BladeRunner Docs

This folder contains build and implementation documentation for the BladeRunner linear stage controller.

## Documents

- [Hardware Assembly Guide](hardware-assembly-guide.md): Mechanical parts, purchased components, wiring, and assembly workflow.
- [Implementation Guide](implementation-guide.md): Firmware, GUI, serial protocol, and system behavior.

## Current Scope

These documents reflect the current repository state:

- Arduino Uno R4 Minima controller
- Adafruit TMC2209 breakout in DIR/STEP mode
- StepperOnline 17HS08-1004S motor
- 45 mm software travel limit
- USB-powered Arduino and external 12 V motor supply
